# Waves.

Interactive visualization of sound waves and harmonics.

## Demo

https://waves.nuage.is

## Features

- 5 harmonics visualization (fundamental + 4 harmonics)
- Real-time audio synthesis (Web Audio API)
- Two visualization modes: Fixed (standing wave) / Scrolling (traveling wave)
- Overlay mode to superimpose all waves
- Adjustable frequency (55-440 Hz), visual speed, and volume

## Deployment

Static HTML file. Just serve `index.html`.
